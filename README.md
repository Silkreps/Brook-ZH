# 全球国际工程招标预警系统

生产目标：云端长期运行的内部管理员系统，自动抓取国际工程项目，核验官方来源，筛选中国企业可参与的大型国际公开采购，翻译为中文并通过 Telegram、邮件和站内消息提醒。

## 技术栈
- Next.js：中文后台与 API。
- Supabase：PostgreSQL、认证、RLS、每日备份。
- Vercel：HTTPS 部署与 Cron。
- n8n：外部调度和失败报警编排。
- OpenAI API：官方资料抽取、翻译、分类、风险分析和评分。

## 本阶段已实现
- 仪表盘首页与后台管理入口。
- 三大板块：资格预审、正式招标、前瞻项目。
- 自动运行 API：`POST /api/cron/run`。
- Supabase 数据库结构：项目、官方链接、数据源、运行日志、推送、管理员审计。
- 核心准入规则：国际公开采购、中国企业资格、金额门槛、未截止、未授标、未取消、官方链接和真实性。
- Telegram/邮件推送接口。
- Vercel Cron 与 n8n 工作流模板。

## 环境变量
复制 `.env.example` 到云平台环境变量。不得将服务端 Key 暴露给前端。
